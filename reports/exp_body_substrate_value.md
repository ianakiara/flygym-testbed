# Experiment Report: Body Substrate Value

## Setup

This experiment compares bodyless abstract agents against full body/reflex + reduced descending agents to determine whether the preserved body/reflex substrate provides measurable benefits for embodied cognition research.

### Environment
- **Embodied condition**: `FlyBodyWorldEnv` with `FlyBodyLayer` + `SimplifiedEmbodiedWorld`
- **Bodyless condition**: `FlyAvatarEnv` with `BodylessBodyLayer` + `AvatarRemappedWorld`
- **Episode length**: 64 steps
- **Seeds**: 5 (0–4)

## Hypotheses

1. **H1**: The body/reflex substrate improves state persistence (higher autocorrelation of internal state) compared to bodyless agents.
2. **H2**: Embodied agents show stronger history dependence (higher action variance within coarse-state buckets) because the body provides richer temporal dynamics.
3. **H3**: Self/world separation is better in embodied agents because ascending feedback distinguishes self-induced from external sensory changes.
4. **H4**: Stabilisation quality is a meaningful metric only for embodied agents — bodyless agents have synthetic stability.

## Methods

### Controllers tested
- **Reduced descending** (target main condition)
- **Memory-augmented** (explicit memory buffer + recurrent hidden state)
- **Reflex-only** (no higher-level control — body-only baseline)
- **Random** (shuffled control — sanity check)

### Metrics
- Task performance (cumulative return)
- State persistence (lag-1 autocorrelation)
- History dependence (action variance in coarse-state buckets)
- Self/world separation (event↔speed correlation)
- Stabilisation quality (stability feature statistics)
- Seam fragility (brain→body command delta)

### Ablation design
- The bodyless condition IS the ablation: it removes the physical body substrate while preserving the controller and (synthetic) ascending feedback.

## Baselines

| Controller | Role |
| --- | --- |
| Reduced descending | Primary experimental condition |
| Memory controller | Tests whether explicit memory amplifies body substrate value |
| Reflex-only | Tests whether gains come from reflexes alone |
| Random | Negative control — should show no body substrate benefit |

## Caveats

- The simplified embodied world may not capture all real-world body dynamics — results represent a lower bound on body substrate value.
- `BodylessBodyLayer` uses synthetic feedback that may not be fully comparable to real physical feedback.
- Short episode length (64 steps) limits the measurement of long-horizon persistence and history dependence.
- Heuristic controllers may not fully exploit the body substrate — trained controllers could show larger effects.
- The comparison is between different body layers AND different world modes — confounded design.

## Failure cases

- If the bodyless agent matches the embodied agent on ALL metrics, the body substrate adds no measurable value for these tasks and controllers.
- If only the reflex-only controller benefits from embodiment, gains are trivially from low-level reflexes rather than body-substrate-enhanced higher-level control.
- If the random controller shows body substrate benefit, the effect is a property of the environment rather than the controller.

## Next steps

1. Extend to longer episodes (256+ steps) to improve persistence measurement quality.
2. Test with trained controllers (RL-based) that can learn to exploit body substrate.
3. Add perturbation experiments to stress the stabilisation substrate.
4. Separate the body-layer confound from the world-mode confound by testing bodyless body in the simplified world.
5. Measure information-theoretic quantities (transfer entropy from body to brain) to quantify ascending signal value.
